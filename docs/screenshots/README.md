# Screenshots

Place portfolio-quality screenshots of the app here. The top-level
`README.md` references them with paths like `docs/screenshots/01-home.png`.

Suggested set (4 to 6 shots):

| Filename | What to show |
|---|---|
| `01-home.png` | Home screen with a featured quote on the user's theme. |
| `02-themes.png` | Themes / customisation page. |
| `03-widget-config.png` | iOS widget configuration. |
| `04-paywall.png` | Paywall (sells the premium tier). |
| `05-onboarding.png` (optional) | Last onboarding step. |
| `06-tutorial.png` (optional) | First-launch tutorial overlay. |

Format guidance:

- PNG, portrait, 1170×2532 (iPhone 15) or similar.
- Crop the device frame out — flat screens read better in a README.
- Keep file size under ~ 600 KB each (run them through `pngquant` /
  `tinypng` if needed).
